# Test Reports

Timestamped test execution summaries and bug reports.

**Folder naming:** `YYYY-MM-DD_HHMMSS` (e.g. `2026-03-11_223000`).

Each run folder should contain:
- `TEST-REPORT.md` — Components tested, bug counts by severity, resolution status, solutions applied.
- (Optional) Playwright HTML report or screenshots if saved here.

**After execution:** Delete test data from the system as requested in the report.
